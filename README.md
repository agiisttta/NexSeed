# NexSeed 🌱

NexSeed adalah media pembelajaran interaktif tentang simulasi perkecambahan biji.

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JSON Web Token (JWT)

## Cara Menjalankan

Backend

```bash
cd backend
npm install
npm run dev
```

Frontend

Buka

```
frontend/html/login.html
```

atau jalankan menggunakan Live Server.
